# Welcome to My Portfolio

Welcome to my portfolio and consultancy site! This is the first post on this platform, and I'm excited to share my journey with you.

## About This Site

This site is built with cutting-edge web technologies:

- **Next.js 15** with the App Router
- **React 19** for the latest React features
- **Tailwind CSS v4** for modern styling
- **Progressive Web App (PWA)** capabilities for offline support
- **MDX** for blog posts with React components

## What You'll Find Here

### Blog
Technical insights, tutorials, and thoughts on modern web development. I'll be writing about:
- React and Next.js best practices
- Progressive Web Apps
- TypeScript patterns
- Architecture and design decisions

### Portfolio
A showcase of my projects, case studies, and applications. Each project includes:
- Live demos (when applicable)
- Source code links
- Technical deep-dives
- Lessons learned

### Services
Information about my consultancy services, including:
- Web application development
- Architecture consulting
- Performance optimization
- Team training and mentoring

## Progressive Web App Features

This site is a full-featured PWA with:

- **Offline Support**: Read blog posts even without an internet connection
- **Install Prompt**: Add this site to your home screen
- **Push Notifications**: Get notified about new content (opt-in)
- **Fast Loading**: Optimized caching strategies for instant page loads

Try installing this site to your device for the best experience!
